## Открытые

### G-001: HTML inline в endpoints
- **Симптом:** layout и partials повторяются в каждом `public/*.php`.
- **Влияние:** дублирование, риск рассинхрона стилей, сложнее редизайн.
- **Решение:** вынести в `templates/` (layout + header / footer / nav).
- **Приоритет:** средний.

### G-002: Тесты ограничены lint + preflight
- **Симптом:** нет feature-тестов, нет CI.
- **Решение:** PHPUnit / Pest, GitHub Actions с lint + preflight + smoke в контейнере.
- **Приоритет:** средний, после стабилизации SaaS-функционала.

### G-003: Self-service регистрация компаний
- **Симптом:** новую компанию создаёт только superadmin.
- **Решение:** публичная форма + email confirmation + автоматический welcome-flow.
- **Приоритет:** низкий, пока не нужно по продукту.

### G-004: Email-уведомления
- **Симптом:** нет напоминаний клиенту про follow-up отзывы (через месяц, год).
- **Решение:** SMTP-канал + cron-задача `send_review_reminders`.
- **Приоритет:** средний.

### G-005: Замена фото на месте
- **Симптом:** для замены приходится удалить и загрузить заново, теряется метаданные.
- **Решение:** `replace` action в `/photo_upload.php`.
- **Приоритет:** низкий.
