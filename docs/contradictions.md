# Contradictions & Gaps

## X-001: Fallback PDF в HTML под расширением `.pdf`
- **Симптом:** при отсутствии mPDF сохраняется HTML как `.pdf`.
- **Почему плохо:** файл невалидный как PDF.
- **Решение:** вернуть явную ошибку и инструкцию установки mPDF.
- **Приоритет:** высокий.

## X-002: Нет полноценного CRUD пользователей (admin)
- **Симптом:** нет UI для создания монтажников/смены паролей.
- **Решение:** добавить раздел администрирования пользователей.
- **Приоритет:** высокий.

## X-003: Нет полного фото-чеклиста по типам работ
- **Симптом:** только часть шаблонов для кондиционеров.
- **Решение:** заполнить `photo_templates` для MVP scope.
- **Приоритет:** высокий.

## X-004: Отсутствуют CSRF-проверки на GET-удаление
- **Симптом:** удаление фото через GET-ссылку.
- **Решение:** перевести на POST + CSRF token.
- **Приоритет:** высокий.

## X-005: Нет шаблонной системы UI (`templates/`) в коде
- **Симптом:** HTML размазан по endpoint-файлам.
- **Решение:** вынести layout и partials в `templates/`.
- **Приоритет:** средний.
