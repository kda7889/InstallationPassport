# MVP Scope

## Релизный scope (выпущено)

- Авторизация, роли superadmin / admin / installer, изоляция арендаторов.
- Создание монтажа + элементов оборудования.
- Загрузка фото с телефона (JPEG/PNG/WebP, HEIC через Imagick), MIME-валидация, JPEG-нормализация, thumbnail.
- Теги стадий фото: До / В процессе / После / Прочее.
- Генерация PDF с QR и двумя кодами доступа (publik + personal).
- Customer-портал: публичная проверка талона по QR с маскировкой ПДн (152-ФЗ); личный доступ для отзывов.
- Отзывы клиентов: 5 критериев, periods (initial / 1m / 1y / 2y / 3y / custom), модерация админом.
- Лендинг: топ компаний / топ монтажников + поиск гарантийника.
- Управление компаниями (superadmin), приостановка с фактическим закрытием доступа.
- Бренд компании: ИНН, контакты, логотип; logo serving вне webroot.
- Login rate-limit + audit_log + TTL.

## Should-have (для ближайших итераций)

- Предупреждение о пропущенных важных фото перед PDF.
- Замена / редактирование фото и его описания.
- История версий PDF.
- Self-service регистрация компаний (сейчас только superadmin).
- Email-уведомления (request-отзыва через месяц / год).
- PWA / offline draft.

## Не входит в текущий scope

- Биллинг / тарификация компаний.
- Push-уведомления.
- Полноценный мобильный клиент.
