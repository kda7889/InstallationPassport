# Mobile-first UX

## Design rules
- Одноколоночные формы.
- Крупные кнопки (thumb-friendly).
- Минимум текстовой плотности.
- Короткие сценарии (до 3-4 экранов на задачу).

## Upload UX
- Кнопка камеры (`capture=environment`) + галерея.
- Видимый прогресс.
- После upload: thumbnail + replace/delete.
