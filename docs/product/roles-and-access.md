# Roles & Access

## superadmin
- `is_superadmin = 1`. На UI обозначается рядом с email.
- Видит и редактирует данные всех компаний.
- Управляет арендаторами через `/companies.php` (создание / отключение).
- Не подпадает под блокировку при приостановке компании.
- Первый активный админ автоматически получает superadmin при миграции с моно-арендной версии.

## admin
- `role = 'admin'`, `company_id` указан, `is_superadmin = 0`.
- Видит и редактирует данные своей компании: монтажи, фото, PDF, пользователей.
- Модерирует отзывы в `/reviews.php`.
- Управляет брендом своей компании в `/company_edit.php`.
- При деактивации компании теряет доступ (логин блокируется, активная сессия рвётся).

## installer
- `role = 'installer'`, `company_id` указан.
- Видит и редактирует **только свои** монтажи в своей компании.
- Создаёт элементы, грузит фото, генерирует PDF в рамках своих объектов.
- При деактивации компании теряет доступ.

## customer (без аккаунта)
- Открывает свой талон через QR из PDF (`customer.php?n=…&c=…`).
- Получает уровень доступа в зависимости от кода:
  - `verification_code` → **public**: ПДн скрыты, отзыв оставить нельзя.
  - `access_token` → **personal**: всё открыто + кнопка «Оставить отзыв».
- Поиск гарантийника через `/` (лендинг) с маскированным просмотром.
