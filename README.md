# МонтажПаспорт (MVP каркас)

Базовый каркас PHP 8 + SQLite для сервиса мобильной фотофиксации монтажных работ и генерации гарантийного PDF.

## Что реализовано в этом каркасе

- Архитектура «один монтаж -> много элементов -> фото по объекту и по элементам».
- Роли `admin` и `installer`.
- SQLite-схема для пользователей, монтажей, элементов, шаблонов фото, загруженных фото и PDF-документов.
- Базовые страницы и точка входа.
- Заготовки модулей безопасности: auth, CSRF, permissions.

## Быстрый старт

1. Установить зависимости (когда будет composer.json): `composer install`
2. Инициализировать БД:
   - создать `storage/database.sqlite`
   - выполнить `database.sql`
3. Настроить web-root на папку `public/`.

## Структура

См. ТЗ: монтаж — это объект работ, внутри которого находятся элементы (кондиционеры/линии/камеры и т.д.), а фото хранятся как на уровне объекта, так и на уровне элемента.
